= `qsh_warn`

----
qsh_warn [<text>...]
----

Prints a warning message to `stderr`.

If at least one `<text>` argument is given, the message will be the
space-separated concatenation of the `<text>` arguments.
Otherwise, the message will default to `"Unknown warning"`.

The message will be prefixed with `"$0: Warning: "` and suffixed with
`"."`.
