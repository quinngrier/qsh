= `qsh_barf`

----
qsh_barf [<text>...]
----

Prints an error message to `stderr` and terminates the script.

If at least one `<text>` argument is given, the message will be the
space-separated concatenation of the `<text>` arguments.
Otherwise, the message will default to `"Unknown error"`.

The message will be prefixed with `"$0: Error: "` and suffixed with
`"."`.

If `qsh_exit_status` is set, it will be used as the exit status.
Otherwise, the exit status will default to 1.
